How Data Deletion Works

Step 1: Request Submission

Users submit a data deletion request through the account settings or by contacting support through email or discord server.

Step 2: Verification

We verify the request by confirming user identity through authentication methods.

Step 3: Processing the Request

The system removes all personal data from databases and storage servers.

Step 4: Confirmation

Users receive an email confirming the data deletion has been completed.
